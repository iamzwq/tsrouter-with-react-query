# React + TypeScript + Vite Boilerplate

[中文](./README.md)

### Core Technologies

- **React**: JavaScript library for building user interfaces
- **TypeScript**: Adds static type checking to improve code maintainability and robustness
- **Vite**: Fast build tool with quick cold start and hot reload capabilities. Optimized build configuration.

### Routing

- **@tanstack/react-router**: Next-generation routing solution (File-based routing + Automatic code splitting)

### Data Fetching

- **@tanstack/react-query**: Manages and caches asynchronous data with a clean API and efficient data handling
- **@tanstack/react-query-devtools**: Development tools for debugging and monitoring data requests

### Form Handling

- **react-hook-form**: Manages form state and validation with a clean API and high-performance form handling

### UI Component Library

- **@mui/material**: React UI component library based on Material Design, offering rich components and styles
- **@mui/icons-material**: Provides a large collection of Material Design icons

### Code Quality & Tools

- **eslint**: Static code analysis to ensure code adheres to standards
- **prettier**: Code formatter to maintain consistent code style
- **husky & lint-staged**: Automates code checks and formatting before commits

### Additional Libraries

- **@ebay/nice-modal-react**: Declarative modal management
- **react-toastify**: Displays notification messages

### Key Features

1. **Version Control**: Auto-update checks via `useVersionChecker`
2. **Custom Controlled Components**: Combines `react-hook-form` with `MUI` for custom controlled components
3. **Custom Theme**: Deeply customized theme system (theme directory)
